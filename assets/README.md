# Assets Folder

This folder contains images for your portfolio website.

## Required Images:

1. **avatar.jpg** — Your profile photo (recommended size: 400×400px)
2. **favicon.svg** — векторная иконка сайта (`&lt;MT/&gt;`)

## Tips:
- Use high-quality images
- Keep file sizes reasonable (under 500KB each)
- Use JPG or PNG format
