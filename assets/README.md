# Assets Folder

This folder contains images for your portfolio website.

## Required Images:

1. **avatar.jpg** - Your profile photo (recommended size: 400x400px)
   - Place your photo here and name it `avatar.jpg`
   - The site will automatically use this image
   - If no image is found, a placeholder will be displayed

2. **certificate1.png** - First certificate image (recommended size: 400x300px)
3. **certificate2.jpg** - Second certificate image (recommended size: 400x300px)
4. **certificate3.jpg** - Third certificate image (recommended size: 400x300px)

## Tips:
- Use high-quality images
- Keep file sizes reasonable (under 500KB each)
- Use JPG or PNG format
- For certificates, ensure the text is readable
