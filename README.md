# Портфолио Тухтаева Мухаммадсалмона

Современный минималистичный сайт-портфолио для Junior C++ Developer.

## 📁 Структура проекта

```
Portfolio/
├── index.html          # Основной HTML файл
├── style.css           # Стили (Minimalist Dark Theme)
├── script.js           # JavaScript (анимации, интерактивность)
├── assets/             # Папка для изображений
│   ├── README.md       # Инструкция по изображениям
│   ├── avatar.jpg      # Ваше фото (добавьте своё)
│   └── certificate1.png # Сертификат (добавьте свой)
└── README.md           # Этот файл
```

## 🚀 Как использовать

1. **Откройте сайт**: Просто откройте `index.html` в браузере
2. **Добавьте свои фото**: Поместите свои изображения в папку `assets/`
   - `avatar.jpg` - ваше фото (рекомендуется 400x400px)
   - `certificate1.png` - сертификат Skillbox (добавьте свой)
3. **Настройте контакты**: Измените email и Telegram в `index.html` при необходимости
4. **Добавьте свои проекты**: Замените или добавьте ссылки на GitHub

## ✨ Особенности

- **Минималистичный дизайн**: Чистый dark theme с синим акцентом
- **Полностью адаптивный**: Работает на всех устройствах (desktop, tablet, mobile)
- **Анимации**: Smooth scroll, fade-in эффекты, hover анимации
- **Без фреймворков**: Чистый HTML, CSS, JavaScript
- **SEO готов**: Мета-теги для поисковых систем
- **Профессиональный контент**: Реальные проекты, образование, навыки

## 📱 Секции сайта

1. **Hero Section**: Имя, титул, кнопки навигации, фото
2. **About Me**: Подробное описание разработчика, технологии
3. **Skills Section**: Навыки в виде тегов (C++, Unreal Engine, PostgreSQL, etc.)
4. **Certificates**: Сертификат с большим размером (А4 формат)
5. **Projects**: Реальный проект с ссылкой на GitHub
6. **Contact**: Email, Telegram, GitHub профиль

## 🎨 Кастомизация

### Цвета
Измените CSS переменные в `style.css`:
```css
:root {
    --accent: #3b82f6;  /* Акцентный цвет (синий) */
    --bg-primary: #0d0d0d;  /* Основной фон */
    /* ... */
}
```

### Навыки
Редактируйте теги в секции `#skills` в `index.html`

### Проекты
Отредактируйте карточку проекта в секции `#projects` в `index.html`

### Образование
Редактируйте информацию в секции `#education` в `index.html`
